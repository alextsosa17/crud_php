# Proyecto CRUD PHP

Este es un proyecto CRUD simple hecho en PHP con MySQL. A continuación, te explico cómo ponerlo en funcionamiento.

## Requisitos

- **XAMPP**: Para ejecutar Apache y MySQL. Descárgalo de [aquí](https://www.apachefriends.org/es/index.html).

## Pasos para ejecutar el proyecto

1. **Instalar XAMPP** y **ejecutar Apache y MySQL** desde el panel de control.
2. **Crear la base de datos**:
   - Abre phpMyAdmin en [http://localhost/phpmyadmin](http://localhost/phpmyadmin).
   - Ejecuta el archivo `create_database.sql` para crear la base de datos y las tablas necesarias.
3. **Coloca el proyecto en la carpeta `htdocs`** de XAMPP.
4. **Accede al proyecto** en tu navegador en [http://localhost/crud_php](http://localhost/crud_php).

## Notas

- El código JavaScript está repetido, lo hice rápido en una mañana, así que disculpen. Funciona, pero hay cosas por mejorar.
- El proyecto es totalmente funcional para agregar, editar y eliminar registros.
